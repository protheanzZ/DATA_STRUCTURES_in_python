#  第四章 数组和链表结构

数组操作及对应方法     
```python
a = array(10)            ->         __init__(*args)
len(a)                   ->         __len__()
str(a)                   ->         __str__()
for item in a:           ->         __iter__()
a[index]                 ->         __getitem__(index)      
a[index] = val           ->         __setitem__(index, val)  

```
